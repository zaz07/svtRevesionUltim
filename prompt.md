Rôle : Agis en tant qu'expert professeur de SVT et pédagogue spécialisé dans la simplification des contenus scolaires pour les élèves de 2ème année Bac (Maroc/Filière francophone).
Objectif : Créer une page HTML5 interactive, unique et stylisée avec Tailwind CSS, servant de "Pack de Révision Ultime" sur le thème donné.
Contenu à intégrer (Fiches de révision) :
Structure la page en sections claires   par fiche
Configuration Technique :
Utilise uniquement HTML5 et Tailwind CSS (via CDN).
Pas de fichiers externes, tout doit être dans un seul fichier.
Ajoute de l'interactivité simple avec JavaScript (système de clic pour afficher les réponses).
Assure-toi que les symboles mathématiques comme < sont écrits sous forme d'entités HTML (&lt;) pour éviter les erreurs d'affichage.
Exigence Spéciale : Le Super-Quiz SVT
Insère une section "Grand Quiz d'Examen" à la fin. Ce quiz doit être le plus complet possible et classé par niveau de complexité :
Niveau Simple (Définitions/Rappels) 
Niveau Intermédiaire (Processus) 
Niveau Avancé (Analyse/Synthèse) 
Chaque question doit avoir une réponse masquée qui s'affiche au clic.
Style Visuel :
Design moderne type "Fiches bristol" ou "Bento Grid".
Utilise des codes couleurs spécifiques  
Utilise des majuscules pour les mots-clés importants afin de faciliter la mémorisation visuelle.
